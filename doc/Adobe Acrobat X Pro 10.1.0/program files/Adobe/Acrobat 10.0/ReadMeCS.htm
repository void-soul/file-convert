系统要求

 

Acrobat X Pro for Windows 要求使用以下配置：

 

・         1.3 GHz 或更高的处理器

・         Microsoft Windows XP Home、Professional 或 Tablet PC Edition，安装 Service Pack 3 for 32-bit（安装 Service Pack 2 for 64-bit）；Windows Server 2003（32 位和 64 位；对于 64 位，需要 Service Pack 2）；Windows Server 2008（32 位和 64 位）；Windows Server® 2008 R2（32 位和 64 位）；Windows Vista Home Basic、Home Premium、Business、Ultimate 或 Enterprise，安装 Service Pack 2（32 位和 64 位）；Microsoft Windows 7 Starter、Home Premium、Professional、Ultimate 或 Enterprise（32 位和 64 位）

・         安装的 RAM 为 512 MB（推荐 1 GB）

・         1.9 GB 的可用硬盘空间（注意：安装程序在安装过程中会创建临时文件，因此在安装过程中所需的实际硬盘空间大于该值。）

・         1,024 x 576 屏幕分辨率

・         Internet Explorer 7 或 8；Firefox 3.5 或 3.6

・         DVD-ROM 驱动器（如果从 DVD 盘安装）

・         视频硬件加速（可选）

对于 Windows Server 2003 SP2 和 Windows XP SP2（64 位），要求安装 Microsoft Hot Fix KB-930627。请参阅：

http://support.microsoft.com/kb/930627

 

 

Acrobat X Standard for Windows® 要求使用以下配置：

・         1,024 x 576 屏幕分辨率

・         Internet Explorer 7 或 8；Firefox 3.5 或 3.6

对于 Windows Server 2003 SP2 和 Windows XP SP2（64 位），要求安装 Microsoft Hot Fix KB-930627。请参阅：

安装说明

 

按照以下说明进行软件全新安装或升级之前的版本。如果正在安装升级，可能需要提供所有权证明。

 

在 Windows 中安装 Acrobat X

 

1. 重新启动 Windows，并关闭所有防病毒软件。

 

2. 开始安装：

  • 如果您购买的是 DVD 版的 Acrobat，请插入相应的 DVD 盘。
  • 如果您购买的是下载版的 Acrobat，请将软件包下载到硬盘驱动器上的某一位置。

 

Adobe 建议您在安装软件之前备份下载包，最好备份到 DVD 盘上。双击 .exe 以解压缩软件包时，将会在计算机上创建

 

<系统驱动器>:\Documents and Settings\<用户>\Desktop\Adobe Acrobat X 文件夹。安装准备就绪后，打开 Adobe Acrobat X 文件夹，然后双击 setup.exe。

 

3. 在 Adobe Acrobat X 安装对话框中，选择语言（如有必要），单击“下一步”。

 

4. 输入程序副本的序列号。每个所购买的单独授权的软件都有一个序列号。该序列号位于 DVD 盘盒的外面；对于通过 Adobe Store 进行的 ESD（电子软件交付）购买，该序列号位于回执页面。对于通过 ESD 购买的单独授权的软件，也可以在以下网址的联机订购历史记录中找到该序列号：http://www.adobe.com/cn。对于通过 Adobe Licensing Website 购买的软件，该序列号可以在以下网址中找到您的记录：http://licensing.adobe.com。

 

5. 选择安装类型：

  • 典型 - 安装 Acrobat、Acrobat Distiller、Adobe Designer（Acrobat X Standard 中不包含）、PDFMaker 的程序文件、系统的 Adobe PostScript 打印机驱动程序、标准增效工具以及附件。
  • 完全 - 安装所有 Acrobat X 功能，需要的磁盘空间最大。
  • 自定义 - 您可以指定要安装的组件。

 

注意：有关组件的说明，请选择“自定义”，然后单击“下一步”。然后选择组件以查看其说明。如果不希望继续自定义安装，请单击“上一步”返回，以选择其它安装类型。

 

6. 单击“下一步”，并按照屏幕上的说明完成安装。

 

7. 激活是软件在启动时由完成的一系列简单、快速的匿名步骤。有关产品激活的详细信息，请访问 Adobe 网站：http://www.adobe.com/cn/products/activation/

在 Windows 中卸载 Acrobat X

 

1. 启动“控制面板”，选择“添加或删除程序”。

 

2. 从显示的应用程序列表中，高亮 Acrobat X，然后按“更改/删除”按钮。

 

3. 将显示一个对话框，要求确认是否删除 Acrobat。选择“是”以卸载产品。

注册
